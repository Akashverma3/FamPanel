API Docs

HTTP Method POST
API URL https://yourdomain.com/api/v2
API KEY Generate your API Key from Account Settings.
Response Format JSON

Add Order

Parameters Description
key Your API Key
action add
service Service ID
link Link to page
quantity Required Quantity
comments (optional) Comments List (Encoded)
runs (optional) Number of Runs
interval (optional) Interval in Minutes

Example Success Response 


{

    "order": 23501

}

Example Error Response 


{

    "error":"Incorrect service type"

}
